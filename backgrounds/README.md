# Background Images

Place PNG files in this directory to use as paper texture backgrounds for generated documents.

## Requirements:
- **Format**: PNG files only
- **Size**: Recommended 1200x1600 pixels (US Letter aspect ratio)
- **Content**: Paper textures, aged paper, document backgrounds
- **Naming**: Any descriptive names (e.g., `aged_paper_01.png`, `parchment_texture.png`)

## Usage:
When generating HTML documents, the system will:
1. Randomly select one PNG file from this directory
2. Copy it to the output directory as `background.png` 
3. Apply it as a background to all generated documents

## Examples:
- Scanned paper textures
- Aged/vintage paper backgrounds  
- Subtle watermark patterns
- Document texture overlays

Add your PNG background files here to make the generated PDFs look like real printed/scanned documents.
